# Common Wrong Explanations - DO NOT USE

## Purpose
Train AI to avoid phrases that harm conversion, medical accuracy, or user trust.

## How AI Uses This
During Explanation Generation:
1. AI identifies root cause (e.g., Performance Anxiety)
2. AI checks table for that root cause
3. AI avoids all ❌ phrases for that condition
4. AI uses patterns from ✅ replacements

## Key Principles for AI

### NEVER Say:
- "Nerves weak ho gaye"
- "Power/mardangi chali gayi"
- "Permanently damaged"
- "Sperm kam = performance problem"
- "Sab dimaag mein hai"
- "Age ho gayi"
- "Hopeless hai"
- "Addiction lag jayegi"

### ALWAYS Say:
- "Reversible hai"
- "Treatable hai"
- "Cause identify karke targeted treatment"
- "Step by step solution"
- "Normal and common"
- "Recovery possible hai"
- "Personalised plan"
- "Doctor guided treatment"
